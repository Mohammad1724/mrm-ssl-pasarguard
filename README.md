# 🚀 مدیریت هوشمند MRM Manager

**MRM Manager** یک جعبه‌ابزار قدرتمند و همه‌کاره برای مدیریت، ایمن‌سازی و مهاجرت پنل‌های **پاسارگارد**، **ربکا** و **مرزبان** است.

---

## ✨ امکانات کلیدی

| قابلیت | توضیحات |
| :--- | :--- |
| **🔐 مدیریت SSL** | دریافت گواهی SSL خودکار برای پنل، نود و لینک‌های اشتراک. |
| **📦 بکاپ و بازگردانی** | بکاپ‌گیری خودکار و ارسال به **تلگرام** (ساعتی/روزانه). |
| **🔄 ابزار مهاجرت** | مهاجرت کامل از **پاسارگارد به ربکا** (تبدیل دیتابیس Timescale به MySQL). |
| **🌐 تفکیک دامنه** | جدا کردن دامنه ادمین از لینک‌های اشتراک (ضد فیلتر). |
| **🎨 مدیریت قالب** | نصب قالب‌های زیبا برای صفحه اشتراک با یک کلیک. |
| **⚡ ویزارد کانفیگ** | ساخت آسان کانفیگ‌های VLESS Reality، TLS و CDN. |
| **🛠 ابزار ادمین** | مدیریت ادمین‌ها، مشاهده لاگ‌ها و کنترل سرویس‌ها. |

---

## 📥 نصب و راه‌اندازی

دستور زیر را کپی و در ترمینال سرور اجرا کنید:

```bash
sudo bash -c "$(curl -sL https://raw.githubusercontent.com/Mohammad1724/mrm-ssl-pasarguard/main/manager/install.sh)"
```

پس از نصب، فقط کافیست بنویسید:
```bash
mrm
```

## 🛠 راهنمای پیشرفته

### 🔄 مهاجرت (پاسارگارد به ربکا)
۱. وارد `mrm` شوید > **Tools & Settings** > **Migration Tools**
۲. گزینه **Migrate to Rebecca** را انتخاب کنید.
۳. تمام! اسکریپت به صورت خودکار:
   - از اطلاعات شما بکاپ می‌گیرد 📦
   - دیتابیس را تبدیل می‌کند (Timescale/SQLite → MySQL) 🔄
   - ربکا را نصب می‌کند 🚀
   - اطلاعات را بازگردانی می‌کند ✅

### 🤖 بکاپ خودکار تلگرام
۱. وارد `mrm` شوید > **Backup & Restore**
۲. گزینه **Telegram Settings** را انتخاب کنید (توکن ربات و آیدی عددی را وارد کنید).
۳. زمان‌بندی دلخواه را انتخاب کنید (مثلاً هر ۶ ساعت).
