## مهاجرت

میخوای از **پاسارگارد** بری **ربکا**؟

```
mrm → Tools → Migration
```

همین. بقیه‌ش با ما:

- ✓ بکاپ کامل میگیره
- ✓ دیتابیس رو تبدیل میکنه  
- ✓ همه چیو منتقل میکنه
- ✓ خراب شد؟ برمیگرده

---

## بکاپ تلگرامی

```
mrm → Backup → Telegram Settings
```

توکن ربات + آیدی چت = بکاپ خودکار هر ۶ ساعت

دیگه نگران از دست رفتن دیتا نباش.
